# Manim Singularity

奇点 IP 专属 Manim 视觉动效库，用于快速构建具有科技感的数学与物理动画。

## 📦 核心组件 (Features)

本项目主要封装了以下核心模块：

1. **视觉主题 (`theme`)**
   - 包含 `NeonTheme` 全局色彩规范。
   - 提供标准化的 UI 组件生成工具（如 Header 等）。

2. **动画引擎 (`SingularityIP`)**
   - 品牌开场动画。
   - 核心图形与场景标题的平滑转场逻辑。

3. **场景基类 (`EllipseBase`)**
   - 预设的科幻风格坐标网格系统。
   - 提供网格的自动化展开与销毁动画片段。
